# oSoWoSo
open Source World Society

## git Projects
Using [subrepo](https://github.com/ingydotnet/git-subrepo)

## Content:
- [Abyss](https://codeberg.org/oSoWoSo/abyss-packages) - Collection of templates for xbps-src and voidlinux
- [bssg](https://codeberg.org/oSoWoSo/BSSG) - Simple static site generator written in Bash
- [clitest](https://codeberg.org/oSoWoSo/clitest) - Portable POSIX shell script that performs automatic testing in Unix command lines
- [SimpleX-themes](https://themes.osowoso.org) - SimpleX Chat Theme Archive
- [vbm](https://vbm.osowoso.org) - XBPS package management helper for Void Linux
- [vut](https://vut.osowoso.org) - Useful tools for working with XBPS and xbps-src on Void Linux
- [xut](https://xut.osowoso.org) - Installs and updates unofficial packages for Void Linux
